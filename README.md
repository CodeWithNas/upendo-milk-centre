# Upendo Milk Centre — Landing Page

A one-page website for **Upendo Milk Centre**, a family-run dairy business in Moshi, Tanzania. Founded in 2006 (formerly known as Amani Milk Centre), Upendo Milk Centre produces affordable, nutritious dairy products — yoghurt, fermented milk (mtindi), butter, mozzarella, and cheddar cheese — for families, restaurants, shops, schools, and institutions across the region.

## What's included

- **`upendo_milk_centre_v2.html`** — the complete landing page: hero section, an interactive product recommendation quiz, product & pricing listings, an "About Upendo" section, testimonials, and a contact/order form. Built as a single self-contained file using only HTML, CSS, and vanilla JavaScript.
- **`Images/`** — product photography (Yoghurt, Mtindi, Butter, Mozzarella, Cheddar Cheese), group range shots, and the brand logo mark used throughout the site.
- **`Upendo milk LOGO.pdf`** — the original source logo file, kept for reference.

## How to view the site

No build step, server, or dependencies are required. Simply open `upendo_milk_centre_v2.html` directly in any modern web browser (double-click the file, or drag it into a browser window).

## Project status

This is a **static front-end build only**. The contact/order form validates input and displays a success confirmation on submit, but it does not currently send data anywhere — there is no backend or email service connected yet. It's intended for front-end demonstration and client review purposes until that integration is added.

Some product details and contact information (phone number, WhatsApp, email, physical address, business hours, social media links) are currently placeholders, clearly marked in the page, pending real business information from Upendo Milk Centre.
